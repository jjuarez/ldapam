# LDAPAM

    LDAP Attribute Manipulator

## Installation

Add this line to your application's Gemfile:

    gem 'ldapam'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ldapam

## Usage

    $ ldapam help
